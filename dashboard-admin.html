Automatización Extracción de Instagram

Para evitar registrar los datos manualmente (Seguidores, Likes, Alcance) cada semana, existen dos vías principales que se pueden implementar para automatizar el Asesor IA de Marketing.

Opción 1: Instagram Graph API (La vía oficial)

  • Cómo funciona: Conexión directa a los servidores de Meta.
  • Pros: 100% estable, gratuita, y oficial.
  • Requisitos: La cuenta debe ser "Business" y estar vinculada a una Página de Facebook. Requiere validación en Meta Developers.

Opción 2: Terceros (Ej. Apify / RapidAPI)

  • Cómo funciona: Uso de servicios especializados que extraen datos públicos (scraping) eludiendo bloqueos.
  • Pros: Coste 0€ en plan gratis, implementado en 10min, sin configurar Facebook, funciona con perfiles normales.
  • Requisitos: Crear cuenta gratuita en el servicio, obtener una API Key y guardarla en el backend.
Próximos pasos:
Revisa estas opciones con la Dirección de Crosti. Cuando se escoja la opción preferida, se activará un "Cron Trigger" que registrará los datos automáticamente todos los lunes sin necesidad de usar el formulario manual.
